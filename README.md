<h1><strong>Sobre o projeto</strong></h1>
<p>O site FastBuy foi feito com o obetivo do App FastBuy ter um meio de download. Este projeto foi o meu TCC (Trabalho de conclusão de curso) a ideia no geral do projeto 
  era resolver um problema que tinha em minha escola, filas kilométricas para comprar lanches no intervalo. Sabendo disso eu e meu grupo resolvemos criar um projeto para resolver esse problema. </p>
